# Hero Composition Module

Use this prompt layer when the main challenge is not aesthetic selection, but building a strong opening screen with clear hierarchy and immediate focal control.

## Intent
Define the structure and visual priorities of the hero section before detailing components or styling.

## Prompt Template
```text
Apply this hero composition module.

Design the hero section with these structural rules:
- The first screen must establish a single clear focal point
- The headline, support copy, CTA, and supporting visual/data element must have a defined hierarchy
- Do not center everything by habit
- Use composition intentionally: split layout, asymmetry, edge alignment, framing, or directional flow where appropriate
- The hero should communicate what the product is, why it matters, and what action the user should take
- Supporting elements must reinforce the message, not compete with it
- The first screen should feel designed, not assembled from common landing page defaults

Deliver:
- Hero section code
- Short explanation of the composition logic
```

## Structural Checklist
- One primary focal element
- Clear reading order
- CTA visible without visual shouting
- Supporting media or proof used with purpose
- Desktop and mobile hierarchy both feel deliberate

## Notes
If the hero becomes too generic, add:

```text
Reject default centered marketing hero patterns. Build a composition with stronger tension, framing, and directional hierarchy.
```
