# Grid Rhythm Module

Use this prompt layer when the output needs stronger alignment, section rhythm, and spacing discipline across a page or interface.

## Intent
Force the model to think in grid logic, spacing systems, and compositional rhythm instead of ad hoc section stacking.

## Prompt Template
```text
Apply this grid rhythm module.

Structural rules:
- Define a clear grid before styling the interface
- Use alignment consistently across sections
- Build a repeatable spacing rhythm instead of random gaps
- Section transitions should feel intentional, not accidental
- Vary density only when hierarchy or content demands it
- Preserve clean edge alignment and predictable structural logic
- Make the layout easy to scan at a glance
- Mobile layout should preserve rhythm, not just stack blocks vertically

Deliver:
- Page or section code
- Short explanation of grid, spacing scale, and section rhythm
```

## Output Criteria
- Repeated sections should feel related
- Spacing should communicate hierarchy
- The interface should feel calmer and more controlled because of structure

## Notes
If the layout feels loose, add:

```text
Tighten alignment, reduce arbitrary spacing variation, and make the section rhythm more systematic.
```
