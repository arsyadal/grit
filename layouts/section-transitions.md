# Section Transitions Module

Use this prompt layer when a page has multiple sections and the challenge is making the flow feel intentional instead of stitched together.

## Intent
Control how one section leads into the next so the page has momentum, rhythm, and narrative continuity.

## Prompt Template
```text
Apply this section transitions module.

Structural rules:
- Each section should feel connected to the next through spacing, framing, contrast, or compositional logic
- Avoid pages that look like isolated blocks stacked without relationship
- Use transitions to control pacing: compression, expansion, pause, escalation, or emphasis
- Preserve a sense of forward movement through the page
- Let contrast between sections feel intentional, not accidental
- Repeated section types must still feel rhythmically coherent
- Mobile layout should preserve transition logic, not flatten all section boundaries into sameness

Deliver:
- Page or section code
- Short explanation of the transition and pacing logic
```

## Output Criteria
- The page should feel guided from top to bottom
- Section boundaries should be clear without feeling rigid or mechanical
- Transitions should reinforce hierarchy and narrative flow

## Notes
If the page feels choppy, add:

```text
Strengthen the pacing between sections. Use more deliberate shifts in density, framing, and spacing rhythm so the page flows instead of stacks.
```
