# Swiss Editorial Module

Use this prompt after injecting `core/grit.instruction`.

## Intent
Generate a Swiss-inspired editorial interface with disciplined grids, typographic authority, restrained color, and high information clarity.

## Prompt Template
```text
Create a Swiss editorial-style interface for [brand/product/page type].

Visual direction:
- Swiss typography
- Editorial restraint
- Grid-driven composition
- Intelligent asymmetry where useful

Requirements:
- Strong typographic hierarchy must carry the design
- Use a disciplined grid with visible alignment logic
- Favor clean planes, sharp spacing rhythm, and structured whitespace
- Keep the palette minimal and controlled
- Avoid playful startup energy
- Avoid decorative effects that dilute clarity
- Use contrast, alignment, and scale more than ornament
- The result should feel rigorous, literate, and art-directed

Typography rules:
- Headlines should feel assertive and intentional
- Body text should be highly readable
- Use tracking, leading, and weight with precision
- Avoid generic default type ramps

Layout rules:
- Strong edge alignment
- Clear modular sections
- A primary focal area within the first screen
- Mobile adaptation must preserve hierarchy and discipline

Deliver:
- Full component or page code
- Short explanation of the grid, type, and spacing logic
```

## Output Criteria
- The design should feel ordered without becoming sterile
- The layout should make the information easier to scan, not harder
- The visual system should feel credible enough for publishing, finance, culture, or technology contexts

## Notes
If the output becomes too plain, add:

```text
Increase editorial tension through stronger scale contrast, more decisive framing, and clearer page rhythm.
```
