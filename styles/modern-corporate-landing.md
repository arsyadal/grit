# Modern Corporate Landing Module

Use this prompt after injecting `core/grit.instruction`.

## Intent
Generate a modern corporate landing page that feels premium, restrained, and credible without collapsing into generic startup UI.

## Prompt Template
```text
Create a modern corporate landing page for [company/product].

Goals:
- The page should feel premium, clear, and trustworthy
- Avoid generic startup gradients and interchangeable SaaS sections
- Use a disciplined layout system with strong alignment
- Emphasize clarity, hierarchy, and information confidence
- The visual tone should be modern corporate, not playful consumer-app UI

Design constraints:
- Refined typography with a strong headline-to-body contrast
- Limited color palette with one controlled accent color
- Spacious but not empty
- Radius, borders, and shadows must be subtle and systematic
- Hero section should establish authority immediately
- Data, proof, or operational credibility should appear early
- CTA should be visible but not loud in a cheap way
- Mobile layout must remain composed and executive-looking

Sections to include:
- Hero
- Credibility or proof strip
- Product or service overview
- Feature or capability grid
- Client/testimonial/proof block
- Final CTA

Deliver:
- Full landing page code
- Brief explanation of layout logic, type system, and color approach
```

## Output Criteria
- The result should feel art-directed and credible
- The page must avoid template-marketplace sameness
- Each section should contribute to trust, clarity, or conversion

## Notes
If the model becomes too sterile, add:

```text
Increase tension and distinction through stronger typography, clearer asymmetry, or more decisive section framing.
```
