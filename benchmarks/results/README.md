# Benchmark Results

Store only lightweight, representative benchmark summaries in this folder.

Good candidates:

- one markdown summary per benchmark round
- a small number of supporting screenshots
- links to external artifacts if the raw output is large

Avoid turning this folder into a dump of all generated experiments.

Recommended naming:

- `2026-04-15-b2b-security-round-01.md`
- `2026-04-16-dashboard-round-01.md`

Sample included in this repo:

- [`2026-04-15-landing-page-b2b-security-sample.md`](/Users/macintoshhd/Documents/grit/benchmarks/results/2026-04-15-landing-page-b2b-security-sample.md:1)

If benchmark output becomes large, keep the summaries here and move raw artifacts elsewhere.
