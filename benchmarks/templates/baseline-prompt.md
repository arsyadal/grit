# Baseline Prompt Template

Use this as the control prompt.
Do not inject GRIT modules here.

```text
Create a responsive [screen/component] for [product].

Product context:
[describe what the product does]

User goal:
[describe what the user needs to achieve]

Requirements:
- [requirement 1]
- [requirement 2]
- [requirement 3]

Tech stack:
[React / Next.js / Tailwind / etc.]

Constraints:
- [constraint 1]
- [constraint 2]
- [constraint 3]
```

## Rule

Keep the baseline prompt competent and fair.
Do not intentionally make it weak.
