# Sample Baseline Prompt

Use this prompt as the control condition for the `landing-page-b2b-security` benchmark task.

```text
Create a responsive landing page for a B2B security platform using React and Tailwind CSS.

Product context:
The product helps enterprise IT and security teams monitor identity risk, endpoint exposure, and policy compliance.

User goal:
Convince security leaders to trust the product and request a demo.

Requirements:
- hero section
- product value explanation
- trust-building proof points
- feature overview
- CTA for demo booking

Tech stack:
React + Tailwind CSS

Constraints:
- desktop and mobile support
- prioritize clarity and trust
- output complete usable frontend code
```

## Why This Is A Fair Baseline

- it is competent, not intentionally weak
- it includes the same product context and requirements as the GRIT prompt
- it does not use any GRIT modules or special execution rules
