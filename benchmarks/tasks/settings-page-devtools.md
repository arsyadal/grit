# Task: Developer Tools Settings Page

## Product Context

The product is a developer platform with settings for API keys, environments, notifications, and team access.

## User Goal

Make complex settings manageable and easy to scan.

## Deliverable

Generate a responsive settings page.

## Requirements

- section navigation or structure
- API key management area
- environment configuration block
- notification controls
- team access or permissions area

## Tech Stack

React + Tailwind CSS

## Constraints

- should feel structured and high-signal
- avoid cluttered admin defaults
- preserve scanability across dense content
