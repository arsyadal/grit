# Task: HR Onboarding Form

## Product Context

The product is an HR platform used by mid-size companies to onboard new employees.

## User Goal

Help HR staff complete employee setup quickly and accurately.

## Deliverable

Generate a responsive onboarding form screen.

## Requirements

- personal information fields
- role and department fields
- document upload area
- action footer

## Tech Stack

React + Tailwind CSS

## Constraints

- high clarity over decoration
- should reduce perceived complexity
- mobile form flow should remain usable
