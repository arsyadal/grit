# Task: Supply Chain Analytics Dashboard

## Product Context

The product helps operations teams monitor supply chain risk, shipment delays, and supplier performance.

## User Goal

Help operators understand system status at a glance and identify issues quickly.

## Deliverable

Generate a responsive analytics dashboard screen.

## Requirements

- summary metrics
- alert or risk module
- chart or trend area
- table or operational detail section

## Tech Stack

React + Tailwind CSS

## Constraints

- should feel information-dense but readable
- avoid generic admin dashboard defaults
- mobile should stay legible, not just collapsed
