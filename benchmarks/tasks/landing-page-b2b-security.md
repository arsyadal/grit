# Task: B2B Security Landing Page

## Product Context

The product is a B2B security platform for enterprise IT teams.
It monitors identity risk, endpoint exposure, and policy compliance.

## User Goal

Convince security leaders to trust the product and request a demo.

## Deliverable

Generate a responsive marketing landing page.

## Requirements

- hero section
- product value explanation
- trust-building proof points
- feature overview
- CTA for demo booking

## Tech Stack

React + Tailwind CSS

## Constraints

- desktop and mobile support
- avoid generic startup landing page patterns
- prioritize hierarchy and trust
